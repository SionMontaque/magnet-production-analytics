# Magnet Production Throughput Analytics

A data-driven project analysing manufacturing throughput, downtime, and quality in a superconducting magnet production process. Includes data analytics, visualisation, and predictive modelling to identify bottlenecks and support operational decision-making.

**Tech Stack:** Python (Pandas, Numpy, Matplotlib), SQL, PowerBI, Scikit-learn  
**Context:** Inspired by Siemens Healthineers Magnet Technology transition to new Bicester facility

## Key Objectives
- Analyse operational KPIs
- Identify performance bottlenecks
- Build regression model for delay prediction
- Create PowerBI dashboard for stakeholders
